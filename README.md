# sumbagteng-project-dashboard

Next.js dashboard for tracking Sumbagteng telecom project SLA status, reporting, maps, and real-time monitoring.

## Features

- Project and SLA monitoring dashboard.
- Reporting-oriented views.
- Map and real-time monitoring foundation.
- Telecom operations workflow support.

## Tech Stack

- Next.js
- TypeScript
- React
- SQLite

## Getting Started

```bash
npm install
npm run dev
```
